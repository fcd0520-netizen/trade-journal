## プロジェクトについて

Trade Journalは、売買記録を残すだけのアプリではありません。

市場環境・市場テーマ・重要イベント・心理状態・ルール遵守・振り返りを記録し、

**投資家として成長すること**

を目的としたジャーナルです。

---

## 開発理念

- 相場を分析する前に、自分を分析する。
- 勝ちよりも、良い意思決定を評価する。
- 勝ちパターンには寿命がある。
- 市場環境が変われば、自分もアップデートする。
- 資産配分も投資である。
